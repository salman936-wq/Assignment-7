# 👥 KeenKeeper

## 📌 Description

KeenKeeper is a friendship tracking web app built with Next.js. It helps users track calls, texts, and video interactions with friends and view them in a timeline and analytics chart.

---

## 🚀 Live Links

* Live Site: https://salman-b13-a7.netlify.app/
* GitHub: https://github.com/salman936-wq/Assignment-7

---

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* Recharts
* React Toastify

---

## ✨ Features

* Responsive Navbar and UI
* Friend list from JSON data
* Friend details page
* Call/Text/Video tracking
* Timeline of interactions
* Filter timeline by type
* Pie chart analytics
* Loading state and toast notifications
* 404 page

---

## 📁 Pages

* Home
* Friend Details
* Timeline
* Stats
